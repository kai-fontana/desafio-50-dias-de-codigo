# Desafio de 50 dias de código

Este desafio consiste em realizar diversos desafios durante os 50 dias, contribuindo para o aprendizado e reforço de diversos conceitos da programação com Java e nos permitindo focar em praticar tudo isso.
| Semana 1 | Principais Conceitos | Linguagem |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------|-------|
| <p style="text-align: justify;">Criar um sistema que receba uma lista de números e, a partir dela, permita ao usuário realizar soma, média e descobrir o maior e o menor número. </p> | POO, Scanner, Arrays, Switch Case, Collections e etc. | Java  |
